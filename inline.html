CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once.

CSS, like HTML, is a markup language, and can be written in plain text using any text editor. CSS allows you to change the style of any element or class of elements on a web page, or across an entire website, by making the change in only one place.
